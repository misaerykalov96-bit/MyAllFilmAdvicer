# MyAIFimlAdvicer
## Описание проекта
Мой проект - сайт с мозгами на Python. Сайт умеет рекомендовать фильмы по вводу настроения пользователя.
## Стек 
Я использовал HTML, CSS, Python для разработки данного сервиса. Реализован фулстак сайт, с созданием локального сервера и подключением нейросети к сайту. Использованы библиотеки Flask, transformers, pipeline, hf_xet.
Модель нейросети для генерации названий фильмов - sberbank-ai. Нейросеть для понимания и анализа настроения - blanchefort.
## Установка библиотек
Для установки библиотек вам необходимо ввести следующую строчку кода:
```
pip install -r requirements.txt
```
Или установить библиотеку по отдельности:
```
   pip install Flask
   pip install transformers
   pip install pipeline
   pip install torch
 ```
